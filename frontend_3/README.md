# Paint
Палка, палка, огуречик. Вот и вышел человечек. Рисовать человечков конечно интересно, но еще интересней программировать инструменты для рисования.
В этом задании, займемся реализацией примитивного графического редактора
## Базовый уровень
1. Добавьте возможность рисования обычных линий
2. Задайте толщину линии в 12 единиц
3. Добавьте возможность изменения цвета линий
4. Добавьте возможность очистки холста


## Продвинутый уровень
1. Добавьте возможность рисования кругов
2. Сделайте переключение инструментов рисования: круг или линия
